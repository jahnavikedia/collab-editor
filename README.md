# Real-Time Collaborative Text Editor

A collaborative text editor where multiple users can edit the same document simultaneously with real-time sync, live cursors, and conflict-free editing.

## Tech Stack

- **Backend**: Java 17 + Spring Boot 3
- **Frontend**: React 18 + TypeScript
- **Real-time**: WebSocket with STOMP
- **Conflict Resolution**: CRDT (Conflict-free Replicated Data Type)
- **Database**: PostgreSQL
